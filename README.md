# VK Music Player Pro

## Описание

VK Music Player Pro это бесплатный десктопный плеер с открытым исходным кодом для прослушивания музыки из социальной сети ВКонтакте. Программа не содержит рекламы и предоставляет доступ к вашим аудиозаписям, плейлистам и поиску по всей базе VK. Приложение работает на операционных системах Windows, Linux и MacOS.

Исходный код программы полностью открыт и доступен для изучения, модификации и распространения в соответствии с условиями лицензии GPL v3. Это означает, что вы можете свободно использовать программу, изменять её под свои нужды и делиться своими изменениями с другими пользователями.

## Основные возможности

Программа позволяет слушать все ваши аудиозаписи без ограничений. Реализована полная загрузка музыкальной коллекции с поддержкой пагинации, поэтому вы получите доступ ко всем трекам, а не только к первым ста. Встроенный менеджер плейлистов отображает все созданные вами плейлисты и позволяет прослушивать музыку из них. Поиск работает по всей базе ВКонтакте и возвращает до двухсот результатов по каждому запросу.

Интерфейс программы включает три основные вкладки: Моя музыка, Плейлисты и Поиск. Управление воспроизведением стандартное: кнопки Play, Pause, Next, Previous, ползунок громкости и прогресс-бар. Приложение запоминает состояние воспроизведения и автоматически переключает треки по окончании текущего.

## Установка

Для работы программы необходим Python версии 3.8 или выше. Скачайте исходный код из репозитория или создайте файл vk_music_player.py с кодом программы.

Создайте виртуальное окружение для изоляции зависимостей:
```
python3 -m venv venv
source venv/bin/activate
```
Для Windows используйте команду venv\Scripts\activate.

Установите необходимые библиотеки:
```
pip install PyQt6 vk-api requests Pillow
```
Или используйте файл requirements.txt:
```
pip install -r requirements.txt
```
Запустите программу командой:
```
python vk_music_player.py
```
## Получение токена доступа

Для работы программы требуется токен доступа ВКонтакте. Без него невозможно получить данные о ваших аудиозаписях и плейлистах.

Самый простой способ получить токен - использовать встроенную функцию программы. Нажмите кнопку Войти в VK, затем в диалоговом окне выберите Получить токен в браузере. Браузер откроет страницу авторизации ВКонтакте. Войдите в свой аккаунт и разрешите доступ приложению. После этого скопируйте токен из адресной строки. Токен находится между access_token= и &expires_in. Вставьте скопированное значение в поле ввода программы и нажмите Войти.

При получении токена важно помнить, что он привязывается к вашему IP-адресу. Не используйте VPN или прокси во время получения токена. При смене IP-адреса токен перестаёт работать и требуется получить новый.

Токен сохраняется в файле vk_token.json в папке с программой. При следующем запуске авторизация происходит автоматически.

# Подробнее про проблему VPN

### Природа проблемы

Когда вы получаете токен доступа ВКонтакте, серверы VK запоминают IP-адрес, с которого был выполнен запрос на получение этого токена. Этот IP-адрес привязывается к токену в целях безопасности. При каждом последующем запросе к API ВКонтакте с использованием этого токена сервер проверяет, совпадает ли текущий IP-адрес с тем, который был запомнен при выдаче токена.

Если IP-адрес не совпадает, ВКонтакте блокирует запрос и возвращает ошибку с текстом "User authorization failed: access_token was given to another ip address". Это стандартная мера защиты, которая предотвращает использование украденных токенов на других компьютерах или злоумышленниками из других сетей.

### Как VPN влияет на работу программы

VPN-сервис изменяет ваш реальный IP-адрес на IP-адрес сервера, к которому вы подключаетесь. Это происходит следующим образом. Вы подключаетесь к VPN-серверу, и весь ваш интернет-трафик начинает идти через этот сервер. Для всех внешних сервисов, включая ВКонтакте, ваш IP-адрес становится IP-адресом VPN-сервера.

Если вы получаете токен при включённом VPN, этот токен привязывается к IP-адресу VPN-сервера. Когда вы выключаете VPN и пытаетесь использовать этот же токен с вашего реального IP-адреса, ВКонтакте видит несовпадение и блокирует запрос. Аналогичная ситуация происходит, если вы получаете токен без VPN, а затем включаете VPN и пытаетесь использовать этот токен.

### Практические сценарии

Рассмотрим несколько типичных ситуаций, в которых возникает эта проблема.

Первый сценарий. Вы получаете токен на домашнем компьютере, на котором не используется VPN. Затем вы переносите файл с токеном на рабочий компьютер, который находится в другой сети или использует VPN. При попытке использовать токен на рабочем компьютере ВКонтакте видит другой IP-адрес и блокирует доступ.

Второй сценарий. Вы используете VPN для доступа к заблокированным сайтам или для обеспечения приватности. При включённом VPN вы открываете программу, которая автоматически использует сохранённый токен, полученный ранее без VPN. Сервер ВКонтакте видит IP-адрес VPN-сервера вместо вашего реального IP и отклоняет запрос.

Третий сценарий. Ваш интернет-провайдер использует динамические IP-адреса, которые меняются при каждой перезагрузке роутера или по истечении времени аренды. Если вы получили токен с одним IP-адресом, а при следующем запуске программы ваш IP-адрес изменился, ВКонтакте также может заблокировать запрос, хотя это не связано с VPN.

### Почему такая защита существует

ВКонтакте внедрил привязку токена к IP-адресу для защиты пользователей от кражи аккаунтов. Если злоумышленник перехватит ваш токен, он не сможет использовать его со своего компьютера, потому что IP-адрес будет другим. Это особенно важно для мобильных приложений и веб-сервисов, где токены используются для авторизации.

Такая защита эффективно предотвращает атаки типа "человек посередине" и кражу сессий. Даже если токен будет скомпрометирован, злоумышленник не сможет его применить без доступа к той же сети, из которой был получен токен.

### Как обойти проблему
Существует несколько способов решения проблемы с IP-адресом.

Первый способ. Получайте токен только на том компьютере и в той сети, где вы планируете использовать программу. Если вы используете VPN постоянно, получайте токен при включённом VPN и никогда не выключайте его во время работы программы. Если вы не используете VPN, получайте токен без VPN и никогда не включайте VPN во время работы программы.

Второй способ. Если вы используете динамический IP-адрес, который меняется, вам придётся получать новый токен при каждом изменении IP. Это неудобно, но единственный способ сохранить доступ к музыке. Некоторые интернет-провайдеры предоставляют статические IP-адреса за дополнительную плату, что решает эту проблему.

Третий способ. Используйте авторизацию через логин и пароль непосредственно в программе. Этот способ также может работать, но ВКонтакте часто запрашивает капчу или дополнительное подтверждение при входе с нового устройства. Кроме того, хранение пароля в программе менее безопасно, чем использование токена.

Четвёртый способ. Некоторые прокси-сервисы и VPN-провайдеры предлагают выделенные IP-адреса, которые не меняются. Использование такого сервиса позволяет сохранить постоянный IP-адрес даже при использовании VPN. Однако это платная услуга и не всегда доступна.

## Управление программой

Интерфейс программы разделён на несколько функциональных зон. В верхней части находится панель авторизации с кнопкой входа и индикатором статуса. Там же расположены кнопки для загрузки треков и отладочной информации.

Основное пространство занимают три вкладки. Вкладка Моя музыка отображает все ваши аудиозаписи в виде списка. Двойной клик по треку начинает воспроизведение. Вкладка Плейлисты показывает список ваших плейлистов слева и их содержимое справа. Вкладка Поиск содержит поле ввода запроса и список результатов.

В нижней части программы находится панель управления воспроизведением. Кнопки навигации позволяют переключать треки, кнопка Play запускает или ставит на паузу воспроизведение. Прогресс-бар отображает текущую позицию в треке. Ползунок громкости регулирует уровень звука.

## Горячие клавиши

Для удобства управления предусмотрены горячие клавиши. Клавиша Space запускает или ставит на паузу воспроизведение. Стрелка вправо переключает на следующий трек, стрелка влево возвращает к предыдущему. Стрелка вверх увеличивает громкость, стрелка вниз уменьшает.

## Настройка и хранение данных

Программа хранит токен доступа в файле vk_token.json. При необходимости вы можете удалить этот файл, чтобы сбросить авторизацию. Все данные хранятся локально и не передаются на сторонние серверы.

Логи работы программы выводятся в терминал при запуске из командной строки. Это помогает отслеживать процесс загрузки треков и выявлять возможные ошибки. Для просмотра подробной информации о состоянии программы используйте кнопку Debug.

## Системные требования

Для работы программы достаточно любого современного компьютера с операционной системой Windows, Linux или MacOS. Требуется подключение к интернету для доступа к API ВКонтакте и загрузки аудиофайлов. Программа потребляет около 256 мегабайт оперативной памяти и занимает примерно 50 мегабайт на диске.

Исходный код программы открыт и доступен для модификации. Вы можете адаптировать его под свои нужды или добавить новые функции. При возникновении вопросов или предложений создавайте Issue в репозитории проекта.

### Рекомендации для пользователей

Для стабильной работы программы рекомендуется выбрать один подход и придерживаться его постоянно. Если вы предпочитаете слушать музыку без VPN, отключите VPN перед получением токена и никогда не включайте его во время работы программы. Если вам необходимо использовать VPN для других целей, настройте раздельное туннелирование, чтобы трафик к ВКонтакте не проходил через VPN. Многие современные VPN-клиенты поддерживают эту функцию.

Также рекомендуется создать резервную копию рабочего токена. Если ваш IP-адрес изменится и токен перестанет работать, вы всегда сможете вернуться к предыдущему IP, используя VPN-сервер с тем же адресом или перезагрузив роутер для получения прежнего IP.

Важно понимать, что проблема связана не с программой, а с политикой безопасности ВКонтакте. Программа лишь использует токен, который вы предоставляете, и не может обойти проверку IP-адреса. Это ограничение на стороне ВКонтакте, и изменить его невозможно без нарушения условий использования сервиса.
